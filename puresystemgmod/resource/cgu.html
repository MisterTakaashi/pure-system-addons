I. Le Pure System


Le Pure System est un addon prenant en compte la Réputation et la Réputation RolePlay d'un joueur afin de lui permettre ou non l'accès à un serveur. Les Serveurs ont donc la capacité de régler les niveaux de Reputation et de Reputaion RolePlay admis sur leurs serveurs. Les sanctions misent en place par les administrateurs du serveur feront baisser les deux types de réputations.
Les réputations sont stockées en base de données et sont donc récuparables par tous les serveurs équipés du Pure System, donc un joueur sanctioné vera son accès a d'autres serveurs équipés du Pure System impossible si la réputaion du joueur de concient pas au serveur cible.

II. Les Modalités


Le Pure System :

  • Recupère les données des joueurs. Tel le SteamID et le SteamID64
  • Répertorie les données dans sa base de données et les conserve
  • Répertorie directement un joueur à sa première connection à un serveur équipé du Pure System
  • S'engage a ne jamais transmettre ou utiliser les données à des fins illègales ou nuisibles à l'identité du joueur ni transmettre ces données à des tiers
  • Envoi et récupère des données sur le serveur
  • S'engage a ne jamais utiliser les données d'un serveur à des fin nuisibles pour celui-ci ni transmettre ces données à des tiers
  • Les données concernant un joueur ne sont visibles que par les membres élus par le fondateur du serveur


III. Les Conditions


Pour les Serveurs :

En installant le Pure System sur votre serveur vous acceptez :

  • Que les données des joueurs soient récuperées, tel SteamID,NomRp,Pseudo,SteamID64
  • Que Le PureSystem envoi et récupere des données à/de votre serveur
  • Que les données du PureSystem concernant les joueurs ne sont utilisés que par le PureSystem
  • Que le code source du PureSystem n'est pas autorisé à être commercialisé et/ou transmis par votre serveur
  • Que les conditions qui ont été cités lors de l'entretien avec les dévloppeurs seront réspectées. Ces conditions sont visible sur le Site : Site du Pure System



Pour les Joueurs :

En cliquant sur le bouton "Accepter les CGU" et en rejoignant le serveur vous acceptez :

  • Que votre SteamID et SteamID64 soient récupérés par le PureSystem
  • Que le PureSystem accède à votre profil Steam et a votre liste d'amis


Le Pure System n'est en aucun cas responsable de l'action des administrateurs sur les joueurs
